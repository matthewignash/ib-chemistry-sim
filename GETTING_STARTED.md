# AP vs IB POGIL Comparison Tool — Getting Started

## What You Have

A single, self-contained HTML file that lets you browse, search, filter, and compare all 30 AP Chemistry POGIL activities with their IB Chemistry adaptations.

**File**: `POGIL_AP_IB_Comparison.html` (502 KB)

## Quick Start

1. **Open the file** in any web browser (Chrome, Firefox, Safari, Edge)
   - Simply double-click the file, or
   - Drag it into your browser window

2. **Browse the dashboard** to see all 30 POGILs as cards
   - Each card shows the title, level (SL+HL, HL-only, Enrichment, AP-only), IB topics, and change count

3. **Use the controls** at the top to find what you need:
   - **Search box**: Search by POGIL title (e.g., "Equilibrium", "Acid")
   - **Level filter**: Show only SL+HL, HL-only, Enrichment, or AP-only
   - **Suggested Use filter**: Show only "Direct Use", "Enrichment", "Skip", etc.
   - **Sort by**: Reorder by number, title, or number of changes

4. **Click any card** to drill into the detailed comparison
   - Left side: Original AP text (unchanged)
   - Right side: IB-modified text with terminology swaps and comprehensive notes
   - Top panel: Summary of IB topics, suggested use, terminology changes, and content flags

5. **Navigate between POGILs** using the Previous/Next arrows
   - Navigation respects your current filters

6. **Return to dashboard** by clicking "Back to Dashboard"

## Troubleshooting

**Q: The file won't open**
A: Make sure you're using a modern web browser. If you see text instead of a formatted page, drag the file into your browser window.

**Q: Filters return no results**
A: You may have selected incompatible filters. Try resetting one filter at a time. For example, some HL-only POGILs won't appear if you filter by "Direct Use" (they may show "Direct Use (HL)" instead).

**Q: Text looks small on mobile**
A: The layout is responsive and should adjust automatically. Try pinch-zooming if needed.

**Q: Can I edit or customize the file?**
A: The HTML is readable as text. Advanced users can open it in a code editor to customize colors, add filters, or modify content. The data structure is well-commented.
