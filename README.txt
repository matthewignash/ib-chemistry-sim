AP vs IB POGIL CHEMISTRY COMPARISON APPLICATION
================================================================================

MAIN FILE
---------
POGIL_AP_IB_Comparison.html (502 KB)

This is a complete, self-contained web application for comparing and aligning
all 30 AP Chemistry POGIL activities with the IB Chemistry curriculum.

Simply open the file in any web browser. No installation, no internet, no
external dependencies required.

WHAT IT DOES
------------

1. DASHBOARD VIEW
   - Browse all 30 POGILs as cards
   - Search by title
   - Filter by level: SL+HL, HL Only, AP-Only, Enrichment
   - Filter by suggested use: Direct Use, Enrichment, Skip, etc.
   - Sort by: Number, Title, or Change Count
   - See at-a-glance statistics

2. COMPARISON VIEW (click any card)
   - Adaptation summary showing IB topics, understandings, suggested use
   - Terminology changes (AP → IB) clearly listed
   - Content flags highlighting AP-only sections
   - Side-by-side text comparison:
     * Left panel: Original AP text
     * Right panel: IB-modified text with annotations
   - Navigate with Previous/Next buttons
   - Return to dashboard anytime

3. PRINTING
   - Press Ctrl+P (Windows) or Cmd+P (Mac)
   - Beautiful print layout for reference sheets
   - All controls hidden for clean output

GETTING STARTED
---------------

1. Open POGIL_AP_IB_Comparison.html in any browser
   - Just double-click it, or drag into browser

2. Explore the Dashboard
   - See all 30 POGILs as color-coded cards
   - Read title, level, topics, suggested use, change count

3. Search or Filter
   - Type in search box to find by title
   - Use dropdowns to filter by level or use category
   - Sort by number, title, or number of changes

4. Click a Card to Compare
   - Read the adaptation summary
   - See terminology changes
   - Compare AP text (left) vs. IB text (right)

5. Navigate
   - Use Previous/Next arrows to move through filtered results
   - Click "Back to Dashboard" to return
   - Or press Back in browser

6. Print
   - Use Ctrl+P or Cmd+P for beautiful print layout
