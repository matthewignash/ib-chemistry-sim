## Technical Notes

- **PptxGenJS** is used for programmatic PPTX creation via Node.js. The v2 script (`create_pptx_v2.js`) is the authoritative source. It uses helper functions: `addTopBar()`, `addSlideTitle()`, `makeShadow()`, `makeCardShadow()`.
- **Animated GIFs** are created by `create_animations.py` using matplotlib + Pillow. They auto-play in PowerPoint 365/2019+.
- **Progressive reveal slides** are used as a workaround for PptxGenJS's lack of native animation support — each "step" is a separate slide that builds on the previous one.
- **Image extraction** from PPTX uses Python `zipfile` to read `ppt/media/` contents. Images keep their internal numbering (image1.png, image2.png, etc.) but are prefixed with the source filename.
- **markitdown** (pip package) converts PPTX to markdown text for reading slide content programmatically.
