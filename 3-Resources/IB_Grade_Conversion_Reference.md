| IB Grade | SL % Range |
|----------|------------|
| 7        | 73% - 100% |
| 6        | 61% - 72%  |
| 5        | 50% - 60%  |
| 4        | 38% - 49%  |
| 3        | 26% - 37%  |
| 2        | 15% - 25%  |
| 1        | 0% - 14%   |

## AISC Holistic Rubric (SL and HL same)

| AISC Grade | Minimum % |
|------------|-----------|
| 8          | 85%       |
| 7          | 70%       |
| 6          | 60%       |
| 5          | 50%       |
| 4          | 40%       |
| 3          | 30%       |
| 2          | 20%       |
| 1          | 0%        |

**Notes:**
- Always round to the nearest whole number of marks
- Always present as marks correct, never as percentages
- AISC grades are calculated separately for KU, TT, and C categories
- Each AISC category goes into Schoology as a separate grade entry
